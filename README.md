SKR Metal Suppliers provides high-quality metal washers for industrial and commercial applications. We specialize in:

- **Umbrella Washers** - Durable metal washers for industrial applications
- **Bitumen Washers** - Weather-resistant washers for roofing

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will be available at `http://localhost:8080`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
